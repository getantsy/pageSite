Windows Intrusion Event Analysis

Date: September 2024

In this lab, I conducted a detailed analysis of network traffic using tools such as Wireshark, NetworkMiner, and SNORT, focusing on identifying potential security threats. By examining captured packets, I uncovered evidence of an ARP spoofing attack and a Man-in-the-Middle compromise involving an IP network camera masquerading as the legitimate device on the network. The analysis identified suspicious communications, protocol behaviors, and discrepancies that pointed to malicious activity.

The lab demonstrated my ability to interpret packet data, assess protocol hierarchies, and utilize security tools to detect and analyze network threats. My findings highlighted the importance of implementing robust security measures to mitigate the risks of similar attacks, including device resets, firmware updates, and stronger network access controls.

Malware Event Investigation and Recovery

Date: October 2024

This report details a forensic analysis of a compromised Windows XP system, conducted within the SimSpace Cyber Range environment. The investigation focused on identifying the intrusion vector and tracing attacker activities using a combination of Kali Linux and Windows FLARE VM tools.

Key findings include the identification of multiple open ports (21, 25, 80, 6666/6667) with several vulnerabilities exploited for unauthorized access. Notably, Poisonivy.exe, a well-known remote access trojan (RAT), was detected alongside backdoors such as Netcat and VNC. Logs from FTP, web services, and system events revealed patterns of unauthorized access and data manipulation.

This analysis employed Nmap, Metasploit, and other forensic tools to map the attacker's steps, gather critical evidence, and assess the overall system compromise. The case study emphasizes the importance of comprehensive log analysis and the risks associated with open ports and unpatched services in a networked environment.

Memory Dump Investigation

Date: November 2024

In this lab, I used Volatility to investigate a Windows XP memory dump for signs of compromise. Initial analysis identified normal system processes but also revealed clear indicators of malicious activity, including Hacker Defender (rootkit), Cryptcat (covert communications), Beware IRCD, IROFFER (botnet-related), Poison Ivy (remote access trojan), and unauthorized VNC remote access. Hidden processes were detected through cross-view analysis. Attempts to extract account data were unsuccessful due to file corruption. DLL analysis confirmed suspicious behaviors linked to these threats, indicating the system was compromised, actively communicating externally, and likely part of a botnet.

Cyber Attack Packet Analysis

Date: December 2024

This report investigates a potential security breach using forensic artifacts such as packet captures (PCAP), application logs, and security logs. The analysis, utilizing tools like Wireshark and NetworkMiner, reveals suspicious network traffic between two devices (192.168.134.129 and 192.168.134.132), with multiple open ports and abnormal SMB traffic patterns. The devices exhibit failed SMB logon attempts, encryption for evading detection, and unusual data flow, suggesting a compromise. Signs of data exfiltration, including manipulated TCP flow control and ARP spoofing, further indicate unauthorized access. Given these findings, it is likely that the EternalBlue exploit was used as the attack vector. Overall, the evidence confirms a significant security breach, with potential data theft occurring.
